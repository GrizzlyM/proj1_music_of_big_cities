Я представил свою работу с датасетами жанров музыуи Москвы и Санкт-Петербурга
Выполнение этого проекта стало моей первой приблежненной к реально приблеженной бизнес-задаче работой. В ней я применил навыки работы с датасетами и их предобработкой. 
Проект заключался в выявлении аномалий в данных, выявлении дубликатов и их обработка. Также добавление новых столбцов в датафреймы.
